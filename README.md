# SmartTailor

SmartTailor is a MERN SaaS and marketplace MVP for local tailors and customers.

## Apps

- `client`: React, Vite, Tailwind CSS frontend.
- `server`: Node.js, Express, MongoDB, JWT, Socket.io backend.
- `docs`: Product, architecture, database, and API planning docs.

## Quick Start

```bash
npm run install:all
cp server/.env.example server/.env
cp client/.env.example client/.env
npm run dev
```

Frontend: `http://localhost:5173`

Backend: `http://localhost:5000`
